# 妖精の森の秘密

- ペトラのいる妖精の森が翼妖精でいっぱいになっている理由
- なぜ森の一角に限られているのか
- なぜ容易に近くの森へ移住できないのか
  - 妖精の森に適した条件がある => だからペトラは探しに旅立つ
    - 湧き出る魔力
      - それだけだと奥地に行くほど魔力が濃く多くなるから適さない
    - 翼妖精にとって必要な何か
      - ペトラが旅立つことに問題が生じる
        - その何かを蓄えて出発すれば解決できるかもしれない
        - アリスが放出している

- そもそも翼妖精は何をして暮らしている？
  - 服や楽器が存在することから人族社会と同じような社会基盤がある可能性
    - 服や楽器は魔法的な存在または魔法的に生み出せる
      - 人族の持ち物を模倣して作り出せる魔法
  - 自由に過ごしている
    - 生きていくのに必要な活動が存在しない
      - その代わり森の一部で得られる何かが必要
        - 変質した魔力
          - 濃い
          - 別の存在
            - 特定の木が汲み上げる魔力
              - その木だらけにならないと人族の街が成り立たないのでは具合が悪いのでは？
                - 特別な宿り木がある場所 <= decided
                  - それだと移植できそう
                    - 移植はできないという設定 => 特別な条件が整うと自然に生えるが、整わないところに移植しても枯れてしまう
                  - 翼妖精はその宿り木を好んでそこにしか住み着かないが必要なわけではない <= point
                  - 宿り木の名前や性質
                    - 宿り木が生える条件
                      - 魔力
                      - 魔力以外の何か
                      - 気候にかかわらず生えるので普通の植生とは違う条件
                        - 翼妖精にフィットした風の魔力がある場所 <= decided
                    - 名前
                      - ◯◯の宿り木
                        - 風の宿り木 <= decided
                    - その他
                      - 秋になる実を翼妖精は好む <= point
