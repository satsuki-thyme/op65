# アリスとミリアの関係

- レオナとは対立と和解
- ミリアとは
  - 関係性の変化
    - レオナはお互いを思う方向が反転した
    - ねじれ
      - 師・弟
      - 先達・後進
      - 討つ者・討たれる者
      - 利用・依存の関係
        - 利用する者・される者
        - 支援する者・支援を受ける者
        - 頼る者・頼られる者
        - 貸す者・借りる者
      - 権力・上下関係
        - 支配する者・される者
        - 命じる者・従う者
        - 守る者・守られる者 <= decided
        - 裁く者・裁かれる者
      - 対立・競争の関係
        - 討つ者・討たれる者
        - 戦う者・戦われる者
        - 奪う者・奪われる者
        - 裏切る者・裏切られる者
      - 協力・共生の関係
        - 作る者・使う者
        - 与える者・受け取る者
        - 導く者・共に歩む者
        - 守り合う者同士
